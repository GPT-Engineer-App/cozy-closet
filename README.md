# cozy-closet

An e-commerce store for a clothing brand
- Use Zustand state management for cart functionality
- sort by price
- categories that use tabs to filter
- Title, description, price and image
- checkout page

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/cozy-closet.git
cd cozy-closet
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
